[CISCN 2022 初赛]ez_usb

题目给了提示是键盘协议，键盘数据包的数据长度为8个字节，击键信息集中在第3个字节。所以我们在winshark里面要搜索8个字节长度的数据包 其中2.8.1和2.10.1的数据包字节长度是8个字节 说明键盘给这两个地方键入了值

接着 我们可以对这两个source分别进行提取

从这两个包里面提取出其中的键盘流量 可以在kali里面实现

然后找个脚本还原数据

f_data = input("请输入带处理txt文件的路径：")
shujian = int(input("鼠标流量信息请输入8，键盘流量请输入16："))
f = open(f_data, 'r')

# 整理到out.txt
with open('out1.txt', 'w') as f_out:
    for i in f.readlines():
        s = i.strip()
        # 鼠标流量长度为8 ，键盘流量长度为16
        if len(s) == shujian:
            # 鼠标流量长度为8 ，键盘流量长度为16
            nsl = [s[j:j + 2] for j in range(0, shujian, 2)]
            ns = ":".join(nsl)
            f_out.write(ns)
            f_out.write('\n')

[鹏城杯 2022]Misc_water

打开是一个有密码的压缩包和一张图片 打开图片发现首尾藏了两张图片

将首尾两张图片分离出来

通过观察 发现剩余的十六进制数据是一个倒序的jpg文件 使用脚本可以将图片倒序从而获得完好的图片

with open('1.jpg', 'rb') as f:
    with open('2.jpg', 'wb') as g:
        g.write(f.read()[::-1])

题目提示是盲水印 使用隐性水印工具提取水印

发现密码是：ZC4#QaWbW

压缩包打开是一张图片 提示是png宽高修复

用脚本对其crc值进行校验修改

import zlib
import struct
import argparse
import itertools


parser = argparse.ArgumentParser()
parser.add_argument("-f", type=str, default=None, required=True,
                    help="flag.jpg")
args = parser.parse_args()


bin_data = open(args.f, 'rb').read()
crc32key = zlib.crc32(bin_data[12:29]) # 计算crc
original_crc32 = int(bin_data[29:33].hex(), 16) # 原始crc


if crc32key == original_crc32: # 计算crc对比原始crc
    print('宽高没有问题!')
else:
    input_ = input("宽高被改了, 是否CRC爆破宽高? (Y/n):")
    if input_ not in ["Y", "y", ""]:
        exit()
    else:
        for i, j in itertools.product(range(4095), range(4095)): # 理论上0x FF FF FF FF，但考虑到屏幕实际/cpu，0x 0F FF就差不多了，也就是4095宽度和高度
            data = bin_data[12:16] + struct.pack('>i', i) + struct.pack('>i', j) + bin_data[24:29]
            crc32 = zlib.crc32(data)
            if(crc32 == original_crc32): # 计算当图片大小为i:j时的CRC校验值，与图片中的CRC比较，当相同，则图片大小已经确定
                print(f"\nCRC32: {hex(original_crc32)}")
                print(f"宽度: {i}, hex: {hex(i)}")
                print(f"高度: {j}, hex: {hex(j)}")
                exit(0)

对其高宽进行修改 之后得到flag

[CISCN 2022 初赛]everlasting_night

题目提示是lsb 放在stegsolve中看一下 有一个小细节 是在Alpha2区域藏东西了

发现了一个字符串 f78dcd 383f1b57346

放在kali里面用lsb.py跑一下 将得到的数据放在flag.txt中

拿出来放在010editor中 发现是个rar压缩包（有密码）

题目中提示md5 打开初始的png图片 观察发现 结尾和普通的png图片好像有点不一样 多了一行数据

拿出来md5解密一下：ohhWh04m1

这就是压缩包的解压密码

打开是一个损坏的png图片 使用kali中的gimp工具 先改名为flag.data

得出:horse:flag{607f41da-e849-4c0b-8867-1b3c74536cc4}
